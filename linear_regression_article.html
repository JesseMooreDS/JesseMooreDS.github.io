Understanding Linear Regression

What is Linear Regression?

Linear regression is a fundamental technique in statistics and data science. It helps us model the relationship between two variables by fitting a straight line to the data. In simple terms, linear regression allows us to predict the value of one variable (the dependent variable, y) based on the value of another variable (the independent variable, X).

Why is it important?

Linear regression is widely used in many areas, including economics, biology, and social sciences. It's one of the most basic and easy-to-understand machine learning algorithms. By mastering linear regression, you'll build a strong foundation for understanding more complex models!

Let's Build a Linear Regression Model in Python

We'll use Python and the Statsmodels library to implement linear regression. In this example, we'll look at how marketing spend impacts widget sales, with a simple linear regression model.

Step 1: Import Libraries

Before we start, we need to import some important libraries in Python. These libraries will help us generate data, create plots, and perform statistical analysis.

import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
import statsmodels.api as sm

Explanation: Here we're using numpy for generating data, matplotlib and seaborn for plotting, and statsmodels to build the regression model and perform statistical analysis.

Step 2: Create Sample Data

For this example, let's generate some data where the number of widgets sold (y) depends on the marketing budget (X) in a linear way.

# Create sample data for marketing budget (X) and widgets sold (y)
X = np.linspace(0, 300, 100)  # Marketing budget: 100 data points from 0 to 300
y = (X * 2) + np.random.normal(0, 20, size=X.shape)  # Widgets sold: Linear relation with noise added

Explanation: We're generating data for a marketing budget X ranging from 0 to 300. The number of widgets sold y is roughly twice the marketing budget plus some random noise (to make it more realistic).

Step 3: Visualize the Data

Let's take a look at our data by plotting it. We'll create a scatter plot to visualize the relationship between the marketing budget and widgets sold.

# Scatter plot to visualize the data
plt.scatter(X, y, color='#80deea', edgecolor='black', alpha=0.7)
plt.xlabel('Marketing Budget')
plt.ylabel('Widgets Sold')
plt.title('Marketing Budget vs. Widgets Sold')
plt.grid(True, color='white', linestyle='-', linewidth=0.5)
plt.show()

Explanation: This plot shows the marketing budget on the x-axis and widgets sold on the y-axis. Notice that the data points roughly follow an upward trend, indicating a positive correlation.

Step 4: Create and Fit the Regression Model

Now, we’ll create a linear regression model and fit it to our data. This will help us understand the relationship between X (marketing budget) and y (widgets sold).

# Add a constant to the independent variable (X) to include the intercept term
X_const = sm.add_constant(X)

# Fit the model using Ordinary Least Squares (OLS)
model = sm.OLS(y, X_const)
results = model.fit()

# Print out the results summary
print(results.summary())

Explanation: We add a constant term to X because the linear regression model includes an intercept. Then, we fit the model using OLS (Ordinary Least Squares) and print out a summary of the results, which shows important statistics like the R-squared value, p-values, and coefficients.

Step 5: Visualize the Regression Line

Once we have the model, we can plot the regression line to see how well it fits the data.

# Plot the regression line
plt.scatter(X, y, color='#80deea', edgecolor='black', alpha=0.7)
sm.graphics.abline_plot(model_results=results, color='red')
plt.xlabel('Marketing Budget')
plt.ylabel('Widgets Sold')
plt.title('Regression Line for Marketing Budget vs. Widgets Sold')
plt.grid(True, color='white', linestyle='-', linewidth=0.5)
plt.show()

Explanation: This plot shows the regression line in red. The line best fits the data, minimizing the distance between itself and the data points.

Assumptions in Linear Regression: The LINE Test

For linear regression to produce reliable results, we must check certain assumptions, often called the **LINE** assumptions. These are:

  • L: Linearity - The relationship between the independent and dependent variable is linear.
  • I: Independence - The residuals (errors) of the model are independent of each other.
  • N: Normality - The residuals are normally distributed.
  • E: Equal Variance (Homoscedasticity) - The variance of the residuals is constant across all levels of the independent variable.

We’ll check these assumptions later by looking at residual plots and performing statistical tests.

Conclusion

In this article, we’ve learned how to apply linear regression to predict outcomes based on a single independent variable. We used Python's Statsmodels library to fit the model and interpret the results. By checking the LINE assumptions, we can ensure that our model is valid and reliable.
